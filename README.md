## Getting Started

To run the project on your local machine:

```bash
git clone [https://github.com/asya-ozkol/pokedex-app-asya.git](https://github.com/asya-ozkol/pokedex-app-asya.git)
cd pokedex-app-asya
npm install
npm run dev
